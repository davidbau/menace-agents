Goal: analyze a 48-day human-AI collaboration (porting NetHack C→JS) to identify what works, what doesn't, agent strengths/weaknesses, human strengths/weaknesses, and generalizable techniques for productive LLM-agent use at scale.

**Why:** To create teachable frameworks for others using LLM agents on large projects.
**How to apply:** All analysis work should aim for actionable, generalizable insights — not just project-specific observations.
