David Bau is a researcher analyzing how humans and LLM coding agents collaborate at scale. He led a 48-day project porting NetHack (450k lines of C) to JavaScript using Claude Code agents extensively. Now studying the logs of that collaboration to extract lessons, techniques, and conceptual frameworks for teaching others how to use LLM agents productively on large codebases.
